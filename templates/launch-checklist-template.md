# Launch Checklist Template

**Instructions:** Copy this to a product folder when preparing a new product launch. Check off each item before publishing.

---

## Product file

- [ ] Product file created ([PDF / ZIP / notebook])
- [ ] Product file tested end-to-end by the creator
- [ ] No placeholder text or TODO comments in product file
- [ ] All code examples tested on a clean environment
- [ ] File size is appropriate for Gumroad (under 50MB for Spark plan)

---

## Product folder

- [ ] `product.md` — internal spec complete
- [ ] `gumroad-description.md` — all 12 sections complete, no placeholders
- [ ] `buyer-instructions.md` — prerequisites listed, steps tested
- [ ] `receipt-message.md` — under 300 words, first action achievable in 10 min
- [ ] `customer-onboarding.md` — all 7 sections, over 400 words
- [ ] `cover-prompt.md` — complete AI image prompt
- [ ] `thumbnail-prompt.md` — complete AI image prompt
- [ ] `faq.md` — at least 5 Q&As
- [ ] `upgrade-plan.md` — at least 3 future improvements
- [ ] `update-log.md` — v1.0.0 entry present

---

## Gumroad

- [ ] Product created (or existing product updated)
- [ ] Title matches display_name in `product-control.yaml`
- [ ] Price matches `product-control.yaml`
- [ ] Cover image uploaded
- [ ] Thumbnail uploaded
- [ ] Description pasted from `gumroad-description.md`
- [ ] Receipt message pasted from `receipt-message.md`
- [ ] Product file uploaded
- [ ] Product published (not draft)

---

## System files

- [ ] `product-control.yaml` updated with new product entry
- [ ] `README.md` product table updated
- [ ] `CHANGELOG.md` entry added
- [ ] Version set to 1.0.0

---

## Marketing (day of launch)

- [ ] LinkedIn post drafted and ready
- [ ] Twitter/X post drafted and ready
- [ ] Reddit post drafted (for relevant subreddit)
- [ ] Email to existing subscribers drafted (if applicable)
- [ ] Free checklist updated to mention new product in CTA section

---

**All boxes checked → you are ready to launch.**
