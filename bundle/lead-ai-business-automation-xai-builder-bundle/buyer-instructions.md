## Suggested 7-day plan

**Day 1:** Read the AI Business Playbook. Work through the Part 1 checklist.

**Day 2:** Complete the AI Business Playbook (Parts 2 and 3).

**Day 3:** Open the AI Prompt Vault. Run 5 starter prompts for your specific use case.

**Day 4–5:** Set up and run the Fraud Detection XAI Project Kit notebook.

**Day 6:** Publish your fraud detection project on GitHub using the included README template.

**Day 7:** Set up the Firebase SaaS Starter Kit locally. Run `npm run dev`.

By the end of week 1 you have: a strategic foundation, 5+ working AI prompts, a portfolio project on GitHub, and a local SaaS app running.
