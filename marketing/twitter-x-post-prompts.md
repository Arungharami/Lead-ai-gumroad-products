# Twitter/X Post Prompts — Lead.AI

**Style guide for Twitter/X:**
- One idea per post. Maximum.
- First sentence must work standalone — most people do not click "show more."
- Short threads (3–5 posts) outperform long threads.
- Be specific and useful. Generic tips get scrolled past.

---

## Prompt 1: Short insight post

```
Write a Twitter/X post sharing one specific insight about [TOPIC, e.g., "why 
class imbalance matters more than model choice in fraud detection"].

Requirements:
- Under 240 characters
- Specific, not generic
- Can be understood without clicking anything
- No hashtag spam — zero or one hashtag maximum
- No emojis unless they genuinely add clarity
- Does not mention a product
```

---

## Prompt 2: Preview/teaser post

```
Write a Twitter/X post teasing [SPECIFIC CONTENT from a product, e.g., one 
prompt from the AI Prompt Vault or one SHAP visualization description].

Content to tease: [PASTE THE CONTENT OR DESCRIBE IT]
Product link: [GUMROAD LINK]

Requirements:
- Show the useful content first — the link is secondary
- Under 280 characters for the main post
- If it needs more space, write it as a 2-post mini-thread
- CTA at the end: simple, no urgency language
- 1–2 hashtags maximum
```

---

## Prompt 3: Thread — "X things I learned"

```
Write a 4-post Twitter/X thread about [TOPIC, e.g., "4 things I learned building 
a fraud detection system with SHAP"].

Requirements:
- Post 1: Hook — a surprising or counterintuitive insight
- Posts 2–4: One specific, concrete lesson each
- Each post works standalone if someone only sees one
- Last post: brief close + product link if relevant (honest, not pushy)
- Conversational, not formal
- Total thread: under 1000 characters
```

---

## Prompt 4: Product announcement thread

```
Write a 3-post Twitter/X thread announcing [PRODUCT NAME].

Product: [PRODUCT NAME]
What it is: [ONE SENTENCE]
Who it is for: [SPECIFIC AUDIENCE]
Key value: [THE MAIN THING IT DOES FOR THE BUYER]
Price: [PRICE]
Link: [GUMROAD LINK]

Requirements:
- Post 1: The problem it solves (not the product name)
- Post 2: What the buyer actually gets (specific list)
- Post 3: Link + price + honest CTA
- No: "I am thrilled to announce", "game-changer", fake urgency
- Each post under 280 characters
```

---

## Prompt 5: Engagement question

```
Write a Twitter/X question to start a conversation with [AUDIENCE, e.g., 
"ML engineers" or "founders building AI tools"].

Topic: [SPECIFIC QUESTION, e.g., "What is the hardest part of explaining 
your ML model to a non-technical stakeholder?"]

Requirements:
- Under 200 characters
- Genuinely curious — not rhetorical
- Do not mention a product
- Invites reply with minimal effort
```

---

## Post frequency guide

- 1–3 posts per day is sustainable
- Mix: 1 insight post, 1 preview/tease, 1 question or thread per week
- Reply to every reply within a few hours during launch period
- Do not auto-schedule posts — post them manually so you can engage in real time
