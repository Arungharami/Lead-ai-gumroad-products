# YouTube Shorts Prompts — Lead.AI

**Format:** 30–60 second scripts for screen recordings or talking-head videos.  
**Best approach for early stage:** Screen recordings with voiceover — no camera required.

---

## Prompt 1: SHAP visualization walkthrough (30 seconds)

```
Write a 30-second YouTube Shorts script for a screen recording showing a SHAP 
summary plot from a fraud detection model.

What is shown on screen: A SHAP summary plot with feature names on the y-axis 
and SHAP values on the x-axis.

Script requirements:
- First 3 seconds: hook — what the viewer is about to see and why it matters
- Middle 20 seconds: explain what the plot shows (which features, what the colors mean)
- Last 7 seconds: one sentence on why this matters + where to get the full project
- Conversational voice, not a lecture
- No jargon without instant explanation
- Include on-screen text callouts (describe where they should appear)
```

## Prompt 2: "What is XAI?" explainer (45 seconds)

```
Write a 45-second YouTube Shorts script explaining explainable AI (XAI) for 
beginner developers and AI students.

Format: Talking head or text-animation style.

Script requirements:
- Start with: one real scenario where XAI matters (e.g., "A bank rejected your 
  loan application. You ask why. The AI says: I can't tell you.")
- Explain XAI in 2 sentences max
- Show one concrete example of a SHAP explanation
- End with: "In the Lead.AI Fraud Detection Kit, you build exactly this system"
- Include timing notes (e.g., "[0–5s hook]", "[5–25s explanation]")
- Natural speech — not a bullet-point recitation
```

## Prompt 5: Product launch announcement Short (30 seconds)

```
Write a 30-second YouTube Shorts script announcing [PRODUCT NAME].

Product: [PRODUCT NAME]
What the viewer will see on screen: [DESCRIBE — e.g., product page on Gumroad, 
or a demo of the product]

Script requirements:
- First 5 seconds: the problem this product solves
- Middle 15 seconds: what the viewer gets
- Last 10 seconds: price and where to get it
- No: "I am so excited to announce", fake countdown, fake scarcity
- Honest, direct, specific
```

## Production tips for screen recordings

- Use Loom (free) or QuickTime (Mac) to record your screen
- Record at 1080p — YouTube Shorts needs 1080x1920 (vertical) or 1920x1080 (horizontal)
- Zoom in on the relevant part of the screen so it is readable on a phone
- Use CapCut (free) to add captions and basic edits
- No fancy production needed — clear screen + clear voice is enough
