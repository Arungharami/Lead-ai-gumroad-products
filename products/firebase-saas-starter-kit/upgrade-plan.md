# Upgrade Plan — Firebase SaaS Starter Kit

---

## v1.0.0 (current) — 2026-05-25

Plain HTML/CSS/JS version with Firebase Auth, Firestore, Stripe integration, Firebase Hosting deployment, and deployment checklist.

---

## v1.1.0 (planned)

**Additions:**
- React version of the same kit (Create React App base)
- Video walkthrough: 25-minute setup and deployment screencast
- Better mobile dashboard layout (current version is functional but not optimized for mobile)

**Trigger:** When 10+ buyers have purchased and React demand is confirmed.

---

## v1.2.0 (planned)

**Additions:**
- Next.js version with server-side rendering
- Vercel deployment option (alongside Firebase Hosting)
- Multi-tier Stripe pricing (Free, Pro, Enterprise)
- Team/organization accounts (multiple users per subscription)

**Trigger:** Approximately 4–6 months after v1.1.0.

---

## v2.0.0 (future)

**Major additions:**
- AI SaaS template: pre-integrated OpenAI or Claude API call UI in the dashboard
- Admin panel: user management, subscription management, usage analytics
- New price: $79 (justified by full AI integration and admin panel)

**Trigger:** When sufficient demand exists for a pre-integrated AI SaaS template.

---

## What will never change

- Firebase as the primary infrastructure (not switching to Supabase or AWS)
- Stripe as the payment provider (most buyers are familiar with it)
- The included deployment checklist (always updated to reflect latest Firebase CLI)
