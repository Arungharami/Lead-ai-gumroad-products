# Upgrade Plan — Lead.AI Indie AI SaaS Starter Kit

This living document outlines the development roadmap and planned upgrade cycles for the **Lead.AI Indie AI SaaS Starter Kit**.

---

## v1.0.0 (Current Version) — 2026-06-01
- Clean, no-build vanilla HTML, CSS, and JavaScript templates.
- Configured Email/Password and Google OAuth credentials using Firebase Auth.
- Complete Stripe and Gumroad integration and licensing checkouts guides.
- Responsive public landing pages, pricing grids, dynamic user dashboards, administrative panels, and secure AI generation wrapper components.
- Launch checklist and pre-configured Privacy, Terms, and Refund policies.

---

## v1.1.0 (Planned)
- **Better UI Templates**: Introduce CSS transitions, dark mode variables toggling, and interactive data visualization charts.
- **More AI Tool Examples**: Add pre-configured layout wrappers for text generation, code optimization, and visual image generation modules.
- **More Dashboard Components**: Add tabular data grids, interactive token usage charts, and notification drop-down mockups.

*Trigger: Planned for release within 30 days based on developer feedback.*

---

## v1.2.0 (Planned)
- **Next.js Version Plan**: Complete React-ready components and route routing structures.
- **Supabase Alternative Notes**: Guidance and schema examples for developers who prefer PostgreSQL databases and Supabase Auth.
- **API Proxy Guide**: Detailed code examples and instructions for writing a secure, serverless API proxy using Vercel/Netlify functions or Firebase Cloud Functions to avoid browser API key leakage.

*Trigger: Approximately 60 days after v1.1.0 release.*

---

## v2.0.0 (Major Release)
- **Full Working SaaS App Template**: Complete fully-functional executable codebase.
- **Unified Auth, Payments & Dashboard**: Completely integrated live user subscription locks.
- **Admin Panel & AI Tool Demo**: Live mock dashboard querying secure model completions.
- **Deployment Walkthrough**: Full interactive setup video guide from git repository initialization to live public domain.

*Trigger: Approximately 6 months after v1.2.0 release. Regular price will increase to $79.*
