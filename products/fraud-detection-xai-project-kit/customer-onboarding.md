## Your first 10 minutes

**Goal: Get the environment running and see your first output.**

1. Download and unzip the ZIP file from your Gumroad receipt
2. Open your terminal and navigate to the project folder
3. Run: `pip install -r requirements.txt`
4. Run: `jupyter lab`
5. Open `fraud_detection_xai.ipynb`
6. Run the first 3 cells (Data Loading section)

**Your 10-minute milestone:** The dataset loads without errors and you can see the first data preview table in the notebook.

If you hit an error, check that Python 3.9+ is installed and paste the error message into an email to a.gharami.325@westcliff.edu.

## Your first 1 hour

**Goal: Run the complete notebook and understand the SHAP outputs.**

1. Continue running all cells top to bottom (takes about 15–20 minutes to run)
2. When the notebook finishes, focus on Section 7 (SHAP Explainability)
3. Study the Summary Plot: which 3 features have the highest SHAP values?
4. Study the Waterfall Plot: why was one specific transaction flagged?
5. Write down in plain English what you would tell a client about why the model flagged that transaction

**Your 1-hour milestone:** You can explain in plain English what the SHAP summary plot shows. You know which features your model considers most important for detecting fraud.
